HTML - Where Would You Be Without It?

HTML stands for HyperText Markup Language, and is the standard for creating web pages. Basically, you create a new HTML page using some essential tags for html, head, body. You also need to make sure you close each tag, or the world will end! Ok, nothing that bad will happen, but your page just won't work.

HTML allows you to show just about any content on the Internet (text, images, links, forms, videos) including adding a site that shows funny cat videos (ok, this will be the last time I push cat related material).

Everybody Needs a Head and Body!

Every HTML page needs a Head and Body section to function. The differences are described below.

Head

The head section is generally used to contain information about the page that you don't necessarily see, like the meta keywords/descriptions, title of a page and embedded CSS styles (if not following the recommended convention of using external CSS style sheets). You can also link to other external files here (for example, javascript), as these need to load before the page is displayed. HTML functions from top to bottom in a linear fashion.

The Head tags are:

<head></head>

Body

You enter whatever content you want displayed in the body.

The Body tags are:

<body></body>

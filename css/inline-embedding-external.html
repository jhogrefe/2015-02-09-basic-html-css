Differences Between Inline, Embedded and External Styles

There are three styles to CSS: Inline, Embedded and External. Although all three work, the recommended convention is to use External CSS whenever possible.

Inline CSS Style

Inline CSS styles are written directly in the tag of the HTML document, and only affect the tag they are applied to.

For example:

<p style="color: blue; text-transform: uppercase">

Embedded CSS Style

Embedded CSS styles are styles that are embedded in the head of the HTML document, and only affect the tags on the page they are embedded in.

For example:

<style>
p { color: #00f; }
</style>

External CSS Style

External CSS styles are written in a separate document (.css) and then attached to any HTML document you link to. This makes it easy to make changes across a large amount of files.

For example, in your HTML document head section you add a link like:

<link rel="stylesheet" type="text/css" href="styles.css" />

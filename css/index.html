CSS - Making Your HTML Look Nicer!

Behind every HTML page is a well thought out CSS. CSS stands for Cascading Style Sheets, and the current version available is CSS3. Fr years, HTML design included all styles within each HTML page, making revisions and updates a mindbender, especially when you were dealing with making changes to a large number of HTML files.

CSS breaks out of this convention by allowing you to create a central style sheet (or, in some cases you might have multiple CSS files relating to specific files) that each of your HTML pages uses. This way, you only have to make a change to the CSS file, rather than to every HTML file. This makes it much easier, and gives you more time to look at more funny cat pictures!
